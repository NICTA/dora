scipy.stats.kurtosistest¶

scipy.stats.kurtosistest(a, axis=0)¶

Tests whether a dataset has normal kurtosis

This function tests the null hypothesis that the kurtosis of the population from which the sample was drawn is that of the normal distribution: kurtosis = 3(n-1)/(n+1).

Parameters:
  • a (array) -- array of the sample data
  • axis (int or None) -- the axis to operate along, or None to work on the whole array. The default is the first axis.
Returns:

  • z-score (float) -- The computed z-score for this test.
  • p-value (float) -- The 2-sided p-value for the hypothesis test

Notes

Valid only for n>20. The Z-score is set to 0 for bad entries.
