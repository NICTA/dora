scipy.stats.itemfreq¶

scipy.stats.itemfreq(a)¶

Returns a 2-D array of item frequencies.

Parameters:a ((N,) array_like) -- Input array.
Returns:itemfreq -- A 2-D frequency table. Column 1 contains sorted, unique values from a, column 2 contains their respective counts.
Return type:(K, 2) ndarray

Examples

>>> a = np.array([1, 1, 5, 0, 1, 2, 2, 0, 1, 4])
>>> stats.itemfreq(a)
array([[ 0.,  2.],
       [ 1.,  4.],
       [ 2.,  2.],
       [ 4.,  1.],
       [ 5.,  1.]])
>>> np.bincount(a)
array([2, 4, 2, 0, 1, 1])

>>> stats.itemfreq(a/10.)
array([[ 0. ,  2. ],
       [ 0.1,  4. ],
       [ 0.2,  2. ],
       [ 0.4,  1. ],
       [ 0.5,  1. ]])
